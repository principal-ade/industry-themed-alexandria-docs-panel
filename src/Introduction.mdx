# Alexandria Docs Panel

Welcome to the **Alexandria Docs Panel** documentation!

This panel extension displays and manages repository documentation for Principal ADE applications.

## What Is This Panel?

The Alexandria Docs Panel is a React component distributed via NPM that can be dynamically loaded into panel-compatible host applications. It provides a simple, clean interface for viewing markdown documentation files in a repository.

## How It Works

### Data Requirements

The panel requires the host application to provide:

```typescript
interface MarkdownFile {
  path: string;           // Full file system path
  title?: string;         // Optional document title
  lastModified: number;   // Unix timestamp
}
```

### Actions Used

- `openFile(filePath: string)` - Opens documents when clicked
